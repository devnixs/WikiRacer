# Story 004: Angular Workspace And App Shell

## Goal

Create the frontend foundation and initial design system shell.

## Scope

- Set up the Angular application structure.
- Add the app shell, routing, and feature folders.
- Create base theme tokens for the clean vivid visual direction.
- Add baseline layout primitives and shared UI components.

## Acceptance Criteria

- The Angular app builds and runs.
- Feature folders exist for home, lobby, match, and results.
- A base theme with vivid accent colors and refined typography is present.
- Shared UI primitives can be reused by later stories.
